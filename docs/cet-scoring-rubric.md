# CET-4 / CET-6 主观题评分大纲

本文档整合：
1. **全国大学英语四六级考试大纲** — 写作与翻译 0–15 分制评分细则（官方原文）
2. **CET Skill 蒸馏维度** — 基于 2015–2025 真题趋势的八维度/七维度分析框架

仅用于指导 `src/lib/prompts.ts` 的 AI 批改 prompt 设计，不作为运行时文件。

---

## 一、作文评分细则

### 1. 评分方式

四六级作文统一采用**总体印象评分制**，满分 15 分，划分 5 个固定档次。阅卷先确定各档次标准样卷，阅卷员经培训后对照样卷阅卷。**同档次四六级样卷难度、评分尺度有级别差异。**

### 2. 官方档次表

| 档次 | 分值区间 | 评分描述 |
|------|---------|----------|
| 14 分档 | 13–15 分 | 切题，表意清晰，行文通顺连贯，基本无语言错误，仅零星小失误 |
| 11 分档 | 10–12 分 | 切题，表意清晰，语句连贯，存在少量语言错误 |
| 8 分档 | 7–9 分 | 基本切题，部分内容表意模糊，行文勉强连贯，语言错误数量多，含多处严重错误 |
| 5 分档 | 4–6 分 | 基本切题，表意混乱，连贯性差，大量严重语言错误 |
| 2 分档 | 1–3 分 | 逻辑散乱、思路紊乱，句式破碎，绝大多数句子存在严重错误 |

### 3. Skill 八维度分项评分权重

| 维度 | 权重 | 对应大纲维度 |
|------|------|-------------|
| contentRelevance | 20% | 切题 |
| organization | 15% | 行文结构 |
| coherence | 15% | 连贯性 |
| grammar | 15% | 语言错误程度 |
| vocabulary | 10% | 用词丰富度与准确性 |
| sentenceVariety | 10% | 句式多样性 |
| naturalness | 10% | 通顺与自然度 |
| templateOveruse | 5% | 反模板痕迹（Skill 特有） |

### 5. 字数不足处理

CET-4 少于 120 词、CET-6 少于 150 词，酌情扣分但不阻断提交。

## 二、翻译评分细则

### 1. 评分方式

四六级翻译同样采用**总体印象评分制**，满分 15 分，分 5 档。阅卷规则同作文：先定各档样卷，培训阅卷员对标样卷打分。**同档次四六级译文评分标准存在级别难度差异。**

### 2. 官方档次表

| 档次 | 分值区间 | 评分描述 |
|------|---------|----------|
| 14 分档 | 13–15 分 | 译文精准还原原文含义，行文流畅、结构规整、用词贴切，基本无错，仅有个别小疏漏 |
| 11 分档 | 10–12 分 | 译文大体还原原意，结构较清晰、语句通顺，附带少量语言错误 |
| 8 分档 | 7–9 分 | 勉强译出原文大意，译文勉强连贯，语病繁多且包含部分严重错误 |
| 5 分档 | 4–6 分 | 仅能翻译原文少量内容，译文连贯性差，充斥大量严重语言错误 |
| 2 分档 | 1–3 分 | 只零散译出个别词语 / 短句，整体无法表达原文意思 |

### 3. Skill 七维度诊断框架

| 维度 | 对应大纲维度 |
|------|-------------|
| information completeness | 还原原文含义（信息完整度） |
| meaning accuracy | 用词贴切（准确性） |
| grammar | 语言错误 |
| naturalness | 行文流畅 / 结构规整 |
| terminology & cultural expression | 术语与文化表达 |
| information order & syntax | 语序与句法 |
| Chinglish detection | 中文母语干扰（Skill 特有） |

## 三、批改集成策略

### AI prompt 使用方式

1. 写作/翻译批改 prompt 同时包含官方档次表 + Skill 维度权重 + 15→100 映射
2. 所有反馈必须基于具体错误指出对应维度，而非笼统评价
3. SentenceEdits 必须引用用户原文中的真实句子
4. Strengths/Problems 每项前标注维度名称

### 评分一致性保障

- AI 先列出每个维度的具体证据再打分
- 配合 `answerKey` 机制（阅读/听力）确保客观题一致性
- 双模板（保分 + 提分）提供差异化输出
